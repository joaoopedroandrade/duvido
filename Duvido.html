Como jogar

O Duvido! é um jogo de blefe para 4 jogadores divididos em 2 duplas.

1. A categoria secreta

A cada rodada, um jogador de cada dupla vê uma categoria (ex: "Cidades do estado de São Paulo"). Os parceiros não podem ver!

2. Os lances

Os jogadores que viram a categoria fazem lances alternados, apostando quantos itens seu parceiro consegue falar. Cada lance deve ser maior que o anterior.

3. Duvidar

A qualquer momento, o jogador da vez pode duvidar do lance anterior em vez de aumentar. Isso encerra os lances.

4. O desafio

O parceiro do último lance tem 60 segundos para falar o número apostado de itens da categoria. Se conseguir, ponto para a dupla dele. Se não, ponto para a outra.

5. Vencer

A primeira dupla a atingir o número de pontos definido no setup vence o jogo!
